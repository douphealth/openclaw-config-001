# Macro — Memory Review

## Use When
- Memory is getting noisy
- A week of work has accumulated
- Recent daily notes contain repeated blockers/truths
- Better recall quality needed from semantic memory

## Goal
Keep memory high-signal, current, and easy to retrieve.

## Sequence
1. Read the last 7 daily memory files
2. Identify repeated facts, blockers, and stale contradictions
3. Promote durable state into entity memory files
4. Promote only genuinely stable truths into MEMORY.md
5. Consolidate duplicates instead of copying everywhere
6. Mark stale blockers resolved or replace with current truth
7. Record new reusable workflows in template/note files

## Promotion Targets
- `memory/people/*.md`
- `memory/sites/*.md`
- `memory/projects/*.md`
- `memory/ops/*.md`
- `MEMORY.md`

## Quality Rules
- One current truth beats many historical fragments
- Prefer entity files for active state
- Prefer daily notes for chronology only
- Do not promote transcript residue
- Do not preserve outdated contradictions

## Output Format
- Files reviewed: _list_
- Facts promoted: _what moved where_
- Duplicates consolidated: _count_
- Stale items corrected: _what changed_
- Durable truths added to MEMORY.md: _list_
- Unresolved uncertainty: _anything that needs more info_
