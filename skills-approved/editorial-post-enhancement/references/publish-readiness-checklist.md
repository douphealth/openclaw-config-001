# Publish Readiness Checklist

## Structure
- one optimized overhead title only
- no duplicate H1 inside body
- short keyword-relevant slug
- answer-first intro
- missing reader questions covered

## SEO / AI visibility
- SEO title present
- meta description present
- semantic entities covered naturally
- FAQ / snippet-friendly sections present where appropriate
- references added when trust matters

## UX / design
- article feels intentionally designed
- FAQ section visually stands out
- media supports the content
- visual elements are useful, not decorative junk

## Linking
- internal links are contextual
- anchor text is rich and precise
- links are spread naturally across the article
- important internal links are verified to return 200 when practical
