# HTML Patterns for Editorial Upgrades

Use selectively based on article fit.

## High-value elements
- hero/summary box
- quick answer block
- comparison table
- feature cards
- warning/callout box
- pull quote
- checklist box
- visually distinct FAQ section
- media figure with caption
- embedded video section
- references block

## Rule
Every visual element should improve one of these:
- scanning
- comprehension
- trust
- retention
- conversion support

If it does none of those, skip it.
