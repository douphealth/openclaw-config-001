# FAQ HTML Patterns

Use FAQ styling that makes the section feel valuable, scannable, and visually distinct.

## Pattern goals
- make FAQs easy to spot
- separate each question clearly
- improve scanability on mobile
- avoid a plain wall of H3 + paragraph blocks

## Strong pattern options
### 1. Card stack FAQ
Use a bordered stack of rounded cards with:
- bold question heading
- lighter answer body
- consistent spacing
- subtle alternating backgrounds if needed

### 2. Accordion-style visual treatment
Even when using static HTML, mimic accordion clarity with:
- question row styling
- answer inset block
- icon/marker cue
- strong spacing between items

### 3. Highlighted expert-answer block
For high-value questions:
- add a small label like "Quick answer" or "Best answer"
- use a tinted background
- keep answer concise first, then expand if needed

## Rules
- each FAQ should answer directly in the first 1-2 sentences
- do not use bloated answers when a concise direct answer is stronger
- keep the visual style consistent across the section
- the FAQ section should look intentionally designed, not tacked on
