# Experiment Readout Standards

## Before launch
Lock:
- hypothesis
- primary metric
- guardrails
- minimum runtime
- stopping logic
- implementation owner

## In the readout
Always separate:
- observed result
- statistical confidence
- practical effect size
- business significance
- rollout recommendation

## Common failure modes
- underpowered samples
- instrumentation drift during test
- multiple simultaneous changes
- “winner” declared from temporary volatility
