# Portfolio WordPress Routing

## Woo / digital product pattern
Use for sites like `efficientgptprompts.com`.
Focus on:
- product page clarity
- add-to-cart
- checkout
- payment visibility
- order creation
- delivery/access

## Lead automation pattern
Use for sites like `affiliatemarketingforsuccess.com`.
Focus on:
- signup path
- contact creation
- welcome automation
- CTA progression into service offer

## Service funnel pattern
Use for sites like `gearuptogrow.com`.
Focus on:
- homepage positioning
- service page
- booking/inquiry/payment path
- only later supporting content

## Anti-pattern
Do not apply Woo assumptions to service funnels or service assumptions to Woo paths.
