# Evidence Standard

## Strong
- successful API / AJAX response with expected insert or success markers
- created record confirmed in downstream system
- successful payment / order evidence where safe
- browser-confirmed state when server-side proof is unavailable

## Medium
- validated render path with clear submit wiring
- partial downstream visibility without full end-to-end confirmation

## Weak
- page looks correct
- form exists in DOM
- CTA button is present
- assumptions from guessed endpoints

Always prefer strong evidence when available.
