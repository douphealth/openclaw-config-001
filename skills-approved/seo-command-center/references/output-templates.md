# SEO Output Templates

Standardized output formats for cross-skill SEO workflows.

## SEO Routing Decision Output

```markdown
## SEO Routing Decision
**Request:** [original request summary]
**Scope:** [single-page | site-wide | multi-site]
**Type:** [diagnosis | execution | mixed]

**Route to:** [skill name]
**Rationale:** [1-2 sentences on why this skill]
**Workflow:** [if multi-step, list ordered skills]

**Expected outputs:**
- [ ] [output 1]
- [ ] [output 2]

**Handoff context for next skill:**
- [key finding or constraint]
- [assets in scope]
```

## SEO Audit Summary Output

```markdown
## SEO Audit Summary
**Site:** [domain]
**Audit date:** [date]
**Scope:** [full site | section | single page]

### Critical Issues (fix immediately)
1. **[Issue]** — [evidence] — [recommended fix]
2. ...

### Important Issues (fix within 2 weeks)
1. **[Issue]** — [evidence] — [recommended fix]
2. ...

### Minor Issues (fix opportunistically)
1. **[Issue]** — [evidence] — [recommended fix]
2. ...

### What's Working Well
- [positive finding with evidence]

### Next Steps
1. [ordered action list]
```
