# Output Contracts

## Standard worker output
- Objective completed
- Exact targets worked
- Changes made
- Verification performed
- Blockers / residual risks
- Next best action

## Standard verifier output
- Target tested
- Pass/fail checklist
- Evidence
- Unresolved issues only

## Director synthesis output
- What was delegated
- What completed
- What is verified
- What remains blocked
- What should happen next
