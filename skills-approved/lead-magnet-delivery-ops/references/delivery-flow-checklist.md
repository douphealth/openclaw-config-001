# Delivery Flow Checklist

Use this checklist to verify a lead magnet delivery flow before launch. Check every item with a real test submission.

## Pre-test setup
- [ ] Clear browser cache or use incognito/private window
- [ ] Prepare a fresh test email (Gmail recommended)
- [ ] Have a second test email on a different provider (Outlook, Yahoo)

## Form → Trigger
- [ ] Submit form with test email
- [ ] Confirm form shows success state (not a silent failure)
- [ ] Verify contact record created in CRM/email platform within 60 seconds
- [ ] Confirm correct tags, segments, or list assignment
- [ ] Check no duplicate records on re-submission with same email

## Asset delivery
### If instant page delivery:
- [ ] Thank-you page loads after form submit (no redirect errors)
- [ ] Asset download link is visible and clickable
- [ ] Asset downloads successfully in incognito browser
- [ ] File is the correct version (not a placeholder or old draft)
- [ ] Mobile: link works on phone browser

### If email delivery:
- [ ] Confirmation email sent within 5 minutes of submission
- [ ] Email arrives in primary inbox (not spam/promotions) — test Gmail + Outlook
- [ ] Sender name and subject line are recognizable and match the brand
- [ ] Asset link in email works and downloads the correct file
- [ ] SPF/DKIM pass on email headers (check via mail-tester.com or similar)

### If account/app access:
- [ ] Account created or access granted within expected timeframe
- [ ] Login credentials sent or SSO works
- [ ] Asset is accessible after login
- [ ] No "request access" friction on first visit

## Thank-you page quality
- [ ] Page loads fast (<3 seconds)
- [ ] Confirms what the subscriber will receive (sets expectation)
- [ ] Includes working asset link (even if email delivery is primary — redundancy)
- [ ] Contains a secondary CTA (book call, read related content, start trial)
- [ ] No dead-end messaging ("thanks, you're done")
- [ ] Mobile responsive

## First follow-up (if applicable)
- [ ] Follow-up email sends within planned delay (1–24 hours typical)
- [ ] Email arrives in inbox
- [ ] Email re-links the asset (handles cases where initial delivery failed)
- [ ] Email includes a clear next step, not just "enjoy the guide"
- [ ] Subscriber record updated to reflect follow-up sent

## Cleanup
- [ ] Test submissions removed or tagged as test in CRM
- [ ] Test email addresses unsubscribed from any nurture sequences
- [ ] Flow documented (form ID, trigger, delivery method, thank-you URL, follow-up sequence)
