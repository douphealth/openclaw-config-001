# Audit Order

## 1. Crawlability and indexation
Check:
- robots.txt
- XML sitemap
- noindex / canonicals
- redirect chains / loops
- indexed vs expected pages

## 2. Technical foundations
Check:
- rendering issues
- Core Web Vitals / speed signals
- mobile readiness
- HTTPS consistency
- URL cleanliness

## 3. On-page
Check:
- title tags
- meta descriptions
- H1 / heading structure
- internal links
- image alt text
- keyword / intent alignment

## 4. Content quality
Check:
- depth
- uniqueness
- intent match
- evidence / trust signals

## 5. Authority / ecosystem
Check:
- internal link support
- backlink context if available
- citation / brand consistency where relevant
