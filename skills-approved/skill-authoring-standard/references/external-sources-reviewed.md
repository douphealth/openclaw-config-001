## High-signal principles adopted
- concise skill bodies
- strong trigger descriptions
- progressive disclosure
- one-level-deep references
- deliberate degrees of freedom
- use scripts for deterministic repeated helpers
- test fragile/compliance-heavy skills with realistic scenarios

## Principles intentionally not imported wholesale
- Claude-specific terminology and assumptions
- excessive meta-process and ceremony
- persuasion-heavy framing as a default style
- large amounts of theory inside the main skill

## Workspace stance
External skills are inputs, not source-of-truth.
Distill useful patterns into OpenClaw-native standards before adopting them.
