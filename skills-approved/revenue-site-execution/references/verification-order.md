# Verification Order

For any revenue-path site, verify the shortest path to money or lead creation first.

## Product / checkout site
Check in order:
- add to cart
- checkout
- payment method availability
- order creation
- delivery / access

## Lead-gen / service site
Check in order:
- CTA path
- form submission
- contact creation
- first email / follow-up
- booking or consult path

## Service-positioning site
Check in order:
- homepage offer clarity
- service page
- inquiry / booking CTA
- payment or lead capture handoff
