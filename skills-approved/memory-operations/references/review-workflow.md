# Review Workflow

## Weekly review
1. read the last 7 daily memory files
2. identify repeated truths, repeated blockers, and stale contradictions
3. update entity files with current truth
4. promote only durable cross-session truths into `MEMORY.md`
5. consolidate duplicates
6. mark stale blockers resolved or replace them

## Anti-patterns
- copying the same note into many files
- preserving transcript residue
- promoting temporary issues into long-term memory
- leaving old contradictions unresolved
