# AI Citation Scorecard

## Use
Apply when auditing or rewriting content for GEO/AEO/AI-answer visibility.

## Scoring (1-5 each)
1. Definition Clarity
2. Quotable Statements
3. Factual Density
4. Source Attribution
5. Question Coverage
6. Authority Signals
7. Freshness
8. Structural Extractability

## Interpretation
- 32–40 → AI-ready
- 24–31 → optimize before relying on it
- 16–23 → weak
- 0–15 → not citation-worthy

## Quick Fix Order
1. Add standalone definition
2. Add direct-answer blocks under H2s
3. Add named stats with dates/sources
4. Add FAQ/comparison sections
5. Add author/reviewer/entity trust signals
6. Add unique examples or original observations
