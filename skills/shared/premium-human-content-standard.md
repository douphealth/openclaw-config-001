# Premium Human Content Standard

## Purpose
Force long-form content work to be materially better: more human, more authoritative, more useful, more citable, and more commercially intelligent.

## Mandatory Content Inputs
Before drafting or rebuilding a serious post, define:
- exact topic
- primary keyword
- 3-8 secondary keywords
- search intent
- target audience
- target length range
- tone
- author persona / authority frame

If these are missing, infer them explicitly before writing.

## Mandatory Post Requirements
Every serious post should include:
1. title with primary keyword front-loaded where practical
2. strong meta description
3. hook in first 100 words with immediate payoff
4. direct answer in first 2-3 sentences of major sections
5. logical H2/H3 hierarchy
6. at least one list or table when useful
7. 5+ PAA-style FAQ questions when the topic supports it
8. strong conclusion with key takeaways and next step
9. internal link plan
10. schema recommendation plan
11. FAQ answers that are substantial enough to compete for snippets
12. visual/topic alignment so images and alt text support the actual article intent
13. cluster-correct related guide strategy
14. explicit content-to-noise review if the page shell is CTA-heavy
15. a References section with high-quality, relevant, clickable resources when the topic benefits from external grounding
16. stronger premium HTML composition using safe callouts, visual summary blocks, and cleaner section rhythm where compatible

## SEO / GEO / AEO Rules
- primary keyword in title, intro, at least one H2, slug logic, and conclusion
- secondary terms integrated naturally
- quotable self-contained paragraphs throughout
- definition-first writing where useful
- clear snippet-friendly answers in 40-60 words near question headings
- use FAQs, tables, lists, and comparison blocks intentionally

## Human Quality Rules
- no generic AI filler
- no mystical fluff without explanatory value
- no repetitive templated intros
- include real examples, edge cases, and practical interpretation
- vary sentence rhythm
- prefer confident clarity over padded softness
- write like a serious human editor, not a content mill

## Anti-Cliche Ban List
Avoid phrases like:
- dive into
- landscape
- in today's world
- it's important to note
- game-changer
- revolutionize
- unlock
- harness
- delve

## Output Requirements
For premium article work, produce or embed:
- title tag
- meta description
- slug recommendation
- schema recommendations
- internal linking opportunities
- suggested cluster topics where relevant

## Verification Rule
Do not claim premium quality unless the post is materially improved in:
- clarity
- structure
- usefulness
- search intent fit
- citation/extractability
- reading flow
