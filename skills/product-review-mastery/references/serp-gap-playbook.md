# SERP Gap Playbook (Top-3)

1. Capture top-3 competitor bodies for target query.
2. Run `scripts/serp_gap_top3.py` against current draft.
3. Select up to 20 high-value missing terms/entities.
4. Integrate naturally in relevant sections only.
5. Re-run quality + readability + link checks.

Never force unrelated terms.
