# Email Marketing Command Center

Operate as a command cycle:
1) Diagnose
2) Prioritize
3) Execute micro-batch
4) Verify
5) Report

## Diagnose
- Check deliverability, list health, active automations, and campaign performance.
- Flag P0/P1 risks first (deliverability collapse, compliance gap, broken triggers).

## Prioritize
- Rank by impact x urgency x reversibility.
- Always clear high-risk blockers before optimization work.

## Execute micro-batch
- Apply one scoped, reversible change at a time.
- Preferred order: infrastructure -> automation logic -> copy/CTA -> scheduling.

## Verify
- Verify source truth (settings/state) and live behavior (test path/output).
- Mark partial if either verification layer is missing.

## Report
- Provide concise ledger:
  - changed
  - outcome
  - blockers
  - next actions
  - evidence links

## Quality gates
- Consent/compliance intact.
- No unsupported performance claims.
- No completion claim without evidence.
