# Email Human Copywriting Framework

## Scope Ownership
### Own
- Execute Write natural, high-conversion, human-sounding marketing emails with strict anti-robotic style controls. Use when drafting or rewriting campaigns, newsletters, and sequence emails.
- Apply workflows, gates, and outputs defined in this file and its references/scripts.

### Do Not Own
- Do not override responsibilities of more specific domain skills when one clearly matches.
- Do not claim completion for adjacent systems without their direct verification gates.

## Core rules
1. Write conversationally (read-aloud test required).
2. Use direct "you" voice with grounded specificity.
3. Keep one core idea per email.
4. Keep paragraphs short.
5. Open with a real hook.
6. Remove jargon/filler.
7. Use one clear CTA.
8. Use P.S. only when it adds value.

## Avoid
- generic corporate openers/closers
- AI-ish filler phrases
- empty hype or manipulative urgency

## Quality gate
- human-readability pass
- intent clarity pass
- CTA clarity pass
- trust/compliance pass

## Output
- final draft
- subject variants
- preheader
- CTA variants
