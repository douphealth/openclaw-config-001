# SEO + AEO/GEO Playbook

## Technical first
- Canonical consistency
- Internal linking to topic hubs
- Schema completeness for page type

## Content packaging
- Query-intent title + meta
- Answer-first intro (direct response)
- Scannable sections and comparisons
- FAQ where relevant

## AI visibility
- Explicit definitions and concise answer blocks
- Source-backed claims
- Consistent entity naming across cluster

## GEO (generative engine optimization)
- Include clear recommendations + tradeoffs
- Include context and constraints, not just conclusions
- Keep content current and revision-dated where appropriate
