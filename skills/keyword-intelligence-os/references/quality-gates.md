# Quality Gates

A page passes only if all checks pass.

## Gate 1: Intent
- Primary keyword and page purpose are aligned.
- No mixed-intent confusion in intro/H2 structure.

## Gate 2: Entity coverage
- Core entity defined clearly.
- Supporting/context entities present and accurate.

## Gate 3: Natural language
- No repetitive exact-match stuffing.
- Readability preserved.

## Gate 4: Trust + compliance
- No clickbait/fearbait framing.
- No unsupported certainty claims.

## Gate 5: Internal-link quality
- Links point to true adjacent intent pages.
- Anchor text is descriptive and non-spammy.

## Gate 6: Verification
- Live 200 render check.
- Title/H1/meta reflect planned mapping.
- Status logged with evidence.
