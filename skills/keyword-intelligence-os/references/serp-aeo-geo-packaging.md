# SERP / AEO / GEO Packaging

## Extraction-friendly blocks
- 40-70 word direct answer block under intro.
- Numbered process list for how-to intent.
- Comparison table for alternative intent.
- FAQ with explicit entities and concise answers.

## AI visibility hygiene
- Use clear entity naming and disambiguation.
- Keep claims grounded and specific.
- Prefer short, factual sentences in key summary blocks.
- Add author/reviewer and update signals where possible.

## Snippet safety
- Avoid hype superlatives and unverifiable numeric claims.
- Keep answer blocks directly aligned to query wording.
