# Email Sequence Improvement Engine

## Scope Ownership
### Own
- Execute Monthly sequence audit and iteration system for continuous email performance gains. Use when running ongoing lifecycle optimization across welcome, nurture, and promotional sequences.
- Apply workflows, gates, and outputs defined in this file and its references/scripts.

### Do Not Own
- Do not override responsibilities of more specific domain skills when one clearly matches.
- Do not claim completion for adjacent systems without their direct verification gates.

## Monthly loop
1. Audit by step: open, CTR, CTOR, unsub, conversion.
2. Diagnose weakest step and likely root cause.
3. Design one-variable A/B test.
4. Implement winner and monitor 30-day impact.
5. Scale proven pattern to similar sequences.

## Diagnosis map
- low open → subject/timing/sender
- low CTR → CTA relevance/clarity
- high unsub → expectation mismatch or over-promotion
- metric decay → fatigue/frequency/stale messaging

## Rules
- one variable per test
- adequate sample before conclusions
- changelog required for every update

## Automation hooks
- click tag → branch path
- conversion event → exit sequence
- high engagement → VIP path

## Output
- monthly audit
- test backlog
- promoted winners
- updated playbook
