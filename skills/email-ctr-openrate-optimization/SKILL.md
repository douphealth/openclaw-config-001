# Email CTR + Open-Rate Optimization

## Scope Ownership

- Execute Improve open rate, CTR, CTOR, and unsubscribe outcomes through structured testing and behavioral optimization. Use when optimizing campaign and automation performance.

### Do Not Own

## Core KPIs
- open rate
- CTR
- CTOR
- unsubscribe rate
- revenue/email

## Optimization loop
1. A/B test subjects on every send.
2. Set intentional preheader.
3. Optimize send-time by segment/timezone.
4. Use trusted sender identity (human + brand).
5. Run re-engagement + suppression for inactive contacts.
6. Keep one dominant CTA for conversion emails.
7. Validate mobile-first layout.

## Test discipline
- one variable per test
- sufficient sample before decision
- promote winners, archive losers

## Cadence
- biweekly performance audit
- low-performer triage + next test queue

## Output
- test matrix
- winners + deltas
- next optimization queue
