# Premium Rollout Checklist (Email Mastery)

## Preflight
- Backups available for all touched settings/pages.
- SMTP auth passing (SPF/DKIM/DMARC).
- Plugin versions stable and compatible.

## Placement quality
- Form placement approved before publish.
- Mobile + desktop render sanity check.
- No layout shift or visual clutter.
- Copy quality pass (clarity, trust, no hype).

## Compliance
- Double opt-in enabled.
- Consent checkbox + privacy policy link present.
- Unsubscribe path visible and functioning.

## Automation quality
- Trigger, delay, and exit paths verified end-to-end.
- Tag/list mappings verified for each source.
- Suppression and bounce logic active.

## Evidence pack
- Settings snapshot
- Test send proof
- Trigger test proof
- KPI baseline snapshot
- Rollback instructions
