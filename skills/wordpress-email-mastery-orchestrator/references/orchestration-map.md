# Orchestration Map

## Stage 1 — Infrastructure (Deliverability foundation)
Run:
- wordpress-email-infrastructure-setup

Exit criteria:
- SPF/DKIM/DMARC pass
- SMTP routing + fallback pass

## Stage 2 — List system (Growth foundation)
Run:
- wordpress-newsletter-list-setup

Exit criteria:
- double opt-in + consent logging pass
- segmentation + hygiene pass

## Stage 3 — Welcome conversion engine
Run:
- email-welcome-sequence-automation
- email-human-copywriting-framework

Exit criteria:
- AIM sequence validated
- human copy quality gate pass

## Stage 4 — Performance optimization
Run:
- email-ctr-openrate-optimization
- email-sequence-improvement-engine

Exit criteria:
- A/B process active
- KPI baseline + review cadence active

## KPI core set
- Open rate
- CTR
- CTOR
- unsubscribe rate
- revenue/email

## Delivery cadence
- Week 1: Stages 1-3 deployment
- Week 2+: Stage 4 optimization loop
