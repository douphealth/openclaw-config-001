# On-Page SEO Auditor

## Objective
Diagnose page-level ranking blockers and provide a prioritized fix plan.

## Scope Boundaries
- Include: page content, metadata, headings, URL clarity, internal links, media SEO.
- Exclude: server/CWV/indexing deep dive (technical-seo-checker), backlink analysis.

## Workflow
1. **Collect context**: target keyword set, page URL, target region, business goal.
2. **Audit snippet layer**: title tag, meta description, search intent match, CTR clarity.
3. **Audit structure**: single H1, logical H2/H3 hierarchy, scannability, answer-first sections.
4. **Audit content relevance**: intent fulfillment, entity coverage, originality, trust signals.
5. **Audit optimization hygiene**: keyword placement (natural), alt text, descriptive anchors.
6. **Audit internal linking**: inbound support links, contextual outbound internal links.
7. **Identify gaps vs likely SERP expectations**: missing sections, weak proof, thin depth.
8. **Prioritize fixes**: high-impact/low-effort first, then structural and strategic updates.

## Quality Gates
- Tie each issue to user impact or ranking impact.
- Provide exact fix guidance, not vague recommendations.
- Separate must-fix blockers from optional enhancements.

## Output Artifacts
Return:
1. On-page scorecard by area.
2. Prioritized issue list (P0/P1/P2) with expected impact.
3. Concrete rewrites for title/meta/headings where needed.
4. Verification checklist for post-implementation QA.
