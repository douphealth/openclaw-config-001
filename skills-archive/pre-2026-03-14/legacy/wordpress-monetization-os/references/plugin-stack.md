# Recommended Plugin Stack

| Purpose | Plugin |
|---|---|
| Affiliate link management | ThirstyAffiliates or Lasso |
| Amazon product boxes | AAWP |
| Own affiliate program | AffiliateWP |
| Digital products | Easy Digital Downloads |
| Email capture | Mailchimp for WP or ConvertKit |
| SEO support | Yoast SEO or Rank Math |
| Analytics bridge | MonsterInsights (GA4 ↔ WP) |

## Preflight requirement
Install and validate plugin availability before enabling related actions.
