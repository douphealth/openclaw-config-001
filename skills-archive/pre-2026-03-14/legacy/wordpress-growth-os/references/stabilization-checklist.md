# Stabilization Checklist

1. Homepage/category/money pages HTTP 200 + render sanity
2. Canonical (HTML + header where applicable)
3. robots, sitemap, indexability directives
4. Theme/plugin conflict signs (layout/head output)
5. Internal 404 links on key pages
6. Contrast/accessibility for nav/hero text on key templates
7. Mobile menu + sticky header sanity
8. Backup before any risky write
9. Verify live immediately after write
10. Log evidence and rollback path
