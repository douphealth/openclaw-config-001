# Monetization OS (WordPress)

## Intent tiers
- Tier A: review/vs/best/comparison (buyer intent)
- Tier B: commercial guides (research intent)
- Tier C: informational (feeder intent)

## Page requirements
### Tier A
- Above-fold decision box
- Comparison table
- 2–3 contextual CTA blocks
- End CTA
- Disclosure near first affiliate CTA

### Tier B
- Intro soft CTA
- Mid workflow CTA
- End conversion CTA
- Internal links to Tier A money pages

### Tier C
- Light CTA only
- Strong internal routing to Tier A/B pages

## Affiliate hygiene
Every affiliate link should have:
- `target="_blank"`
- `rel="sponsored nofollow noopener"`
- Nearby disclosure

## Offer architecture
For each money page define:
- Primary offer
- Secondary fallback
- Support offer
- Why this mapping exists

## KPI focus
- CTR on money-page CTAs
- Offer click depth per section
- Money page impressions/clicks/rank trends
- Assisted conversions from feeder pages
