# SERP Gap Playbook (Top-3 + Top-20 Missing Terms)

1. Pull the top-3 ranking pages for the target query.
2. Run `scripts/serp_gap_top3.py` against the draft.
3. Review top 20 missing terms/entities.
4. Integrate naturally in relevant sections only.
5. Re-run quality + readability + link QA.

Never force terms or add empty sections just to include keywords.
