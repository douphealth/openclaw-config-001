# Technical SEO Health

## Scope
- Core Web Vitals monitoring.
- Schema markup (JSON-LD) orchestration.
- Crawl budget management.
- XML Sitemap and Robots.txt optimization.

## Legacy Mapping
- SiteSpeed.md, SchemaMarkup.md, Indexing.md, SitemapRobots.md, TechnicalAudit.md.

## Execution Protocols
1. **Audit:** Conduct a technical sweep of the site for 404s and redirect chains.
2. **Optimize:** Ensure all Tier 1 pages have valid Product/Article schema.
3. **Monitor:** Track CWV metrics and trigger alerts for performance regressions.
